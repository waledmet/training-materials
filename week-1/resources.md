## Cheat Sheets

### Python Quick Reference

**Variables and Data Types**:
```python
# Integer
age = 25

# Float
price = 19.99

# String
name = "Ahmed"

# Boolean
is_student = True

# Check type
type(variable_name)
```

**Operators**:
```python
# Arithmetic
+  -  *  /  //  %  **

# Comparison
==  !=  >  <  >=  <=

# Logical
and  or  not
```

**Control Flow**:
```python
# If statement
if condition:
    # code
elif another_condition:
    # code
else:
    # code

# For loop
for i in range(start, stop, step):
    # code

# While loop
while condition:
    # code
```

**String Operations**:
```python
# Concatenation
"Hello" + " " + "World"

# f-strings
f"My name is {name}"

# Length
len(string)

# Methods
string.upper()
string.lower()
string.strip()
```

**Input/Output**:
```python
# Output
print("text")
print(variable)
print(f"Value: {variable}")

# Input
name = input("Prompt: ")
number = int(input("Number: "))
```

### Terminal Commands

**Windows**:
```bash
dir                # List files
cd folder          # Change directory
cd ..              # Go back
mkdir folder       # Create folder
cls                # Clear screen
python script.py   # Run Python
```

**macOS/Linux**:
```bash
ls                 # List files
cd folder          # Change directory
cd ..              # Go back
mkdir folder       # Create folder
clear              # Clear screen
python3 script.py  # Run Python
```

### Git Commands

```bash
# Configuration
git config --global user.name "Name"
git config --global user.email "email"

# Basic commands
git init           # Initialize repository
git status         # Check status
git add .          # Stage all changes
git add file.py    # Stage specific file
git commit -m "Message"  # Commit changes
git log            # View history

# GitHub
git remote add origin URL  # Connect to GitHub
git push origin main       # Push changes
git pull origin main       # Pull changes
```

## Tips for Using Resources Effectively

### How to Learn from Documentation

1. **Don't try to memorize** - understand concepts
2. **Use Ctrl+F** to search for specific topics
3. **Bookmark frequently used pages**
4. **Try examples yourself** - don't just read
5. **Read error messages** - they often reference docs

### How to Ask Good Questions

When stuck:

1. **Search first** - your question likely exists
2. **Be specific** - include error messages
3. **Show your code** - what you've tried
4. **Explain expected vs actual** - what should happen vs what happens
5. **Format code properly** - use code blocks

**Example Good Question**:
```
I'm trying to calculate the sum of numbers 1-10 but getting a TypeError.

Code:
total = 0
for i in range(10):
    total = total + i

Error:
TypeError: unsupported operand type(s) for +: 'int' and 'str'

Expected: total should be 45
Actual: Getting this error

I checked that i is an integer using type(i). What am I doing wrong?
```

### How to Practice Effectively

1. **Type, don't copy-paste** - muscle memory matters
2. **Start small** - simple programs first
3. **Build incrementally** - add features one at a time
4. **Debug intentionally** - break things to learn
5. **Review others' code** - learn different approaches
