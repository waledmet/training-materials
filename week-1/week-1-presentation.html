The Web Request-Response Cycle

What Happens When You Visit a Website?

  1. You type www.google.com in browser
  2. Browser sends HTTP REQUEST (placing your order)
  3. Server receives request (kitchen gets order)
  4. Server prepares response (kitchen cooks food)
  5. Server sends HTTP RESPONSE back (waiter brings food)
  6. Browser displays webpage (you enjoy your meal!)

Key Terms - Browser vs Server

Browser

  • Lives on YOUR computer
  • Sends requests
  • Displays web pages
  • Examples: Chrome, Firefox, Safari, Edge

Server

  • Lives somewhere else (anywhere in the world)
  • Receives requests
  • Sends back web pages
  • Always running, always waiting
Trainer Note: Use simple language, avoid technical jargon

HTTP Methods

How We Talk to Servers

  • GET - "Please SHOW me something"
    • Example: Loading a webpage, viewing a profile
  • POST - "I want to SEND you something"
    • Example: Submitting a form, creating a post

Restaurant Analogy:

  • GET = "What's on the menu?"
  • POST = "Here's my custom order"
Trainer Note: Don't go too deep - just introduce concepts

HTTP Status Codes

Server Response Codes

  • 200 - OK, everything worked!
  • 404 - Not Found (page doesn't exist)
  • 500 - Server Error (something broke)

Restaurant Analogy:

  • 200 = Your order is ready!
  • 404 = We don't have that dish
  • 500 = Kitchen is on fire!
Trainer Note: Students will see these errors often - normalize them!

Variable Naming Rules

How to Name Variables

Rules (must follow):

  • Start with letter or underscore
  • Can contain letters, numbers, underscores
  • Case sensitive (name ‚â† Name)
  • No spaces or special characters

Best Practices:

  • Use descriptive names: student_age not sa
  • Use lowercase with underscores: first_name
  • Avoid Python keywords: for, if, class

Day 3 - Control Flow

Making Decisions in Code

Why? Programs need to make choices based on conditions

Real Life Example:

If it's raining:
    Take an umbrella
Else:
    Wear sunglasses

In Python:

weather = "rainy"
if weather == "rainy":
    print("Take an umbrella")
else:
    print("Wear sunglasses")
Trainer Note: Start with real-life examples before code

Boolean Expressions

True or False?

Comparison Operators:

5 == 5    # Equal to (True)
5 != 3    # Not equal to (True)
5 > 3     # Greater than (True)
5 < 3     # Less than (False)
5 >= 5    # Greater than or equal (True)
5 <= 3    # Less than or equal (False)

Important: == (comparison) vs = (assignment)

Trainer Note: Show common mistake of using = instead of ==

if Statements

Basic Decision Making

Syntax:

if condition:
    # code to run if True
    # indent is IMPORTANT

Example:

age = 18
if age >= 18:
    print("You can vote!")
Trainer Note: Emphasize indentation importance in Python

if-else Statements

Two Paths

age = 16
if age >= 18:
    print("You can vote!")
else:
    print("You're too young to vote")

Think of it as:

  • if condition is True ‚Üí do this
  • else (if False) ‚Üí do that
Trainer Note: Draw flowchart diagram for visual learners

if-elif-else Statements

Multiple Conditions

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
elif score >= 60:
    print("Grade: D")
else:
    print("Grade: F")
Trainer Note: Explain that it checks conditions in order, stops at first True

Logical Operators

Combining Conditions

AND - Both must be True:

age = 25
has_license = True
if age >= 18 and has_license:
    print("You can drive!")

OR - At least one must be True:

if day == "Saturday" or day == "Sunday":
    print("It's the weekend!")

NOT - Reverses the condition:

if not is_raining:
    print("No umbrella needed")
Trainer Note: Use truth tables if students are struggling

Day 4 - Loops

Why Loops?

Without Loops (repetitive):

print("Hello 1")
print("Hello 2")
print("Hello 3")
print("Hello 4")
print("Hello 5")

With Loops (elegant):

for i in range(1, 6):
    print(f"Hello {i}")
Trainer Note: Show the power of loops with this contrast

for Loops

Repeating Code a Specific Number of Times

Basic Syntax:

for variable in range(start, stop, step):
    # code to repeat

Examples:

# Print 0 to 4
for i in range(5):
    print(i)

# Print 1 to 5
for i in range(1, 6):
    print(i)

# Print even numbers 0 to 10
for i in range(0, 11, 2):
    print(i)
Trainer Note: Explain range(stop), range(start, stop), range(start, stop, step)

while Loops

Repeat While Condition is True

Syntax:

while condition:
    # code to repeat
    # make sure condition becomes False eventually!

Example:

count = 1
while count <= 5:
    print(f"Count: {count}")
    count += 1  # Important! Update the counter

Warning: Infinite loops if you forget to update!

Trainer Note: Show infinite loop example (Ctrl+C to stop)

break and continue

Controlling Loop Flow

break - Exit loop immediately:

for i in range(1, 11):
    if i == 5:
        break
    print(i)
# Output: 1, 2, 3, 4

continue - Skip to next iteration:

for i in range(1, 6):
    if i == 3:
        continue
    print(i)
# Output: 1, 2, 4, 5
Trainer Note: Use real-world examples (break = emergency exit, continue = skip song)
