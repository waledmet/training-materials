## أوراق مرجعية

### مرجع Python السريع

**المتغيرات وأنواع البيانات**:
```python
# Integer (عدد صحيح)
age = 25

# Float (عدد عشري)
price = 19.99

# String (نص)
name = "Ahmed"

# Boolean (قيمة منطقية)
is_student = True

# التحقق من النوع
type(variable_name)
```

**العوامل (Operators)**:
```python
# Arithmetic (حسابية)
+  -  *  /  //  %  **

# Comparison (مقارنة)
==  !=  >  <  >=  <=

# Logical (منطقية)
and  or  not
```

**التحكم في التدفق (Control Flow)**:
```python
# If statement (جملة if)
if condition:
    # code
elif another_condition:
    # code
else:
    # code

# For loop (حلقة for)
for i in range(start, stop, step):
    # code

# While loop (حلقة while)
while condition:
    # code
```

**عمليات النصوص (String Operations)**:
```python
# Concatenation (دمج)
"Hello" + " " + "World"

# f-strings (سلاسل منسقة)
f"My name is {name}"

# Length (الطول)
len(string)

# Methods (طرق)
string.upper()
string.lower()
string.strip()
```

**الإدخال/الإخراج (Input/Output)**:
```python
# Output (إخراج)
print("text")
print(variable)
print(f"Value: {variable}")

# Input (إدخال)
name = input("Prompt: ")
number = int(input("Number: "))
```

### أوامر Git

```bash
# Configuration (الإعداد)
git config --global user.name "Name"
git config --global user.email "email"

# Basic commands (أوامر أساسية)
git init           # تهيئة المستودع
git status         # التحقق من الحالة
git add .          # إضافة جميع التغييرات
git add file.py    # إضافة ملف محدد
git commit -m "Message"  # حفظ التغييرات
git log            # عرض السجل

# GitHub
git remote add origin URL  # الاتصال بـ GitHub
git push origin main       # دفع التغييرات
git pull origin main       # سحب التغييرات
```

## نصائح لاستخدام الموارد بفعالية

### كيفية التعلم من الوثائق

1. **لا تحاول الحفظ** - افهم المفاهيم
2. **استخدم Ctrl+F** للبحث عن موضوعات محددة
3. **ضع إشارة مرجعية على الصفحات المستخدمة بشكل متكرر**
4. **جرب الأمثلة بنفسك** - لا تقرأ فقط
5. **اقرأ رسائل الخطأ** - غالباً ما تشير إلى الوثائق

### كيفية طرح أسئلة جيدة

عندما تكون عالقاً:

1. **ابحث أولاً** - من المحتمل أن سؤالك موجود
2. **كن محدداً** - قم بتضمين رسائل الخطأ
3. **أظهر كودك** - ما جربته
4. **اشرح المتوقع مقابل الفعلي** - ما يجب أن يحدث مقابل ما يحدث
5. **قم بتنسيق الكود بشكل صحيح** - استخدم كتل الكود

**مثال على سؤال جيد**:
```
أحاول حساب مجموع الأرقام من 1-10 لكنني أحصل على TypeError.

الكود:
total = 0
for i in range(10):
    total = total + i

الخطأ:
TypeError: unsupported operand type(s) for +: 'int' and 'str'

المتوقع: يجب أن يكون المجموع 45
الفعلي: الحصول على هذا الخطأ

لقد تحققت من أن i عدد صحيح باستخدام type(i). ما الذي أفعله خطأ؟
```

### كيفية الممارسة بفعالية

1. **اكتب، لا تنسخ وتلصق** - الذاكرة العضلية مهمة
2. **ابدأ صغيراً** - برامج بسيطة أولاً
3. **بنِ بشكل تدريجي** - أضف ميزات واحدة في كل مرة
4. **صحح بشكل مقصود** - اكسر الأشياء للتعلم
5. **راجع كود الآخرين** - تعلم مناهج مختلفة
