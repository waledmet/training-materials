How ListView Works 
class PostListView(ListView):
    model = Post
    template_name = \
      'blog/post_list.html'
    context_object_name = 'posts'
    paginate_by = 5

    def get_queryset(self):
        return Post.objects.filter(
            status='published'
        )

CBVs provide pagination, queryset filtering, and template context automatically.
