## أوراق مرجعية سريعة

### مرجع Django السريع

**وسوم قوالب Django**

```django
{# التعليقات #}

{{ variable }}                  {# عرض متغير #}
{{ variable|filter }}           {# تطبيق مرشح #}

{% if condition %}              {# عبارة if #}
    ...
{% elif other %}
    ...
{% else %}
    ...
{% endif %}

{% for item in list %}          {# حلقة for #}
    {{ item }}
{% empty %}
    No items
{% endfor %}

{% url 'name' %}                {# URL بالاسم #}
{% url 'namespace:name' %}      {# مع مساحة اسمية #}

{% extends 'base.html' %}       {# وراثة القوالب #}
{% block name %}...{% endblock %} {# تعريف كتلة #}

{% include 'partial.html' %}    {# تضمين قالب #}

{% load static %}               {# تحميل الملفات الثابتة #}
{% static 'path/file' %}        {# رابط ملف ثابت #}
```

**مرشحات القوالب الشائعة**

```django
{{ name|lower }}                # أحرف صغيرة
{{ name|upper }}                # أحرف كبيرة
{{ name|title }}                # حالة العنوان
{{ text|truncatewords:10 }}     # تحديد عدد الكلمات
{{ value|default:"N/A" }}       # القيمة الافتراضية إذا فارغ
{{ date|date:"Y-m-d" }}         # تنسيق التاريخ
{{ number|floatformat:2 }}      # تنسيق الأرقام العشرية
{{ list|length }}               # طول القائمة
{{ text|safe }}                 # وسم كـ HTML آمن
```

**أوامر إدارة Django**

```bash
# المشروع والتطبيقات
django-admin startproject name          # إنشاء مشروع
python manage.py startapp name          # إنشاء تطبيق

# خادم التطوير
python manage.py runserver              # تشغيل الخادم
python manage.py runserver 8001         # منفذ مخصص

# قاعدة البيانات (الأسبوع 5)
python manage.py makemigrations         # إنشاء الترحيلات
python manage.py migrate                # تطبيق الترحيلات
python manage.py createsuperuser        # إنشاء مستخدم مدير

# أخرى مفيدة
python manage.py shell                  # shell Python
python manage.py check                  # التحقق من الأخطاء
python manage.py collectstatic          # جمع الملفات الثابتة
```

**أنماط URL**

```python
from django.urls import path

urlpatterns = [
    path('', views.home, name='home'),              # /
    path('about/', views.about, name='about'),      # /about/
    path('post/<int:id>/', views.post, name='post'), # /post/5/
    path('user/<str:username>/', views.user),        # /user/ahmed/
]
```

**أنماط العروض**

```python
from django.shortcuts import render
from django.http import HttpResponse

# استجابة نصية بسيطة
def home(request):
    return HttpResponse("Hello")

# عرض قالب
def home(request):
    return render(request, 'app/home.html')

# مع سياق
def home(request):
    context = {'name': 'Ahmed'}
    return render(request, 'app/home.html', context)

# مع معامل URL
def detail(request, id):
    context = {'id': id}
    return render(request, 'app/detail.html', context)
```

## موارد حل المشاكل

### الأخطاء الشائعة

**"No module named 'django'"**
- الحل: فعّل البيئة الافتراضية، ثم `pip install django`
- الدليل: راجع قسم التثبيت في lessons_ar.md

**"TemplateDoesNotExist"**
- الحل: تحقق من INSTALLED_APPS، تحقق من مسار القالب
- التصحيح: https://docs.djangoproject.com/en/stable/ref/templates/api/

**"Page not found (404)"**
- الحل: تحقق من أنماط URL، تأكد من إعداد urls.py
- دليل التصحيح في قسم حل المشاكل في lessons_ar.md

**صفحات الخطأ:**
- شرح صفحات خطأ Django: https://docs.djangoproject.com/en/stable/ref/views/#error-views
