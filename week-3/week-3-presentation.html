HTTP Protocol

HyperText Transfer Protocol

HTTP Methods

  • GET - Retrieve data
  • POST - Send/create data
  • PUT - Update data
  • DELETE - Remove data

Status Codes

  • 200 - OK (Success)

HTTPS: Secure version of HTTP (encrypted)

HTML Document Structure

Every HTML Page Needs This

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is the content</p>
</body>
</html>
  • <!DOCTYPE html> - Tells browser this is HTML5
  • <head> - Metadata (not visible on page)
  • <body> - Visible content

What is CSS?

Cascading Style Sheets

  • Controls the appearance of HTML elements
  • Separates content (HTML) from presentation (CSS)
  • Makes websites beautiful and professional
Without CSS:

Plain, ugly text with no styling

With CSS:

Beautiful, styled, professional website!

Demo: Show the same HTML page before and after adding CSS

Three Ways to Add CSS

1. Inline (Not Recommended)

<p style="color: red; font-size: 20px;">Red text</p>

2. Internal (In <head>)

<style>
    p { color: blue; }
</style>

3. External (Best Practice) ‚úì

<link rel="stylesheet" href="style.css">

Why external? Separation of concerns, reusability, caching

CSS Selectors

Targeting HTML Elements

/* Element Selector */
p {
    color: blue;
    font-size: 16px;
}

/* Class Selector (reusable) */
.highlight {
    background-color: yellow;
    padding: 10px;
}

/* ID Selector (unique) */
#header {
    background-color: navy;
    color: white;
}
<p class="highlight">Highlighted text</p>
<div id="header">Header content</div>

Best Practices

  • Use DevTools

  • Write clear commit messages

Common Challenges

And How to Fix Them

  • CSS not showing?
    • Check file path in <link>
    • Ensure file is saved
    • Hard refresh: Ctrl+F5
  • HTML tags not closing?
    • Use VS Code auto-close
    • Check indentation
    • Use W3C validator
  • Git authentication failing?
    • Use personal access token (not password)
    • Check token hasn't expired

Questions & Answers

Common Questions

  • Q: Should I memorize all HTML tags?
    A: No! Reference MDN when needed. You'll remember common ones with practice.
  • Q: Which is better, classes or IDs?
    A: Classes for styling (reusable). IDs for unique elements.
  • Q: Do I need to learn Git CLI?
    A: Yes! It's the industry standard. GUI tools are okay later, but learn CLI first.
  • Q: How do I get better at CSS?
    A: Practice, practice, practice! Build real projects, copy designs you like.
