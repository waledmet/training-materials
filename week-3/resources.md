## Quick Reference Cards

### Essential HTML Tags
```html
<!-- Structure -->
<!DOCTYPE html>, <html>, <head>, <body>

<!-- Content -->
<h1>-<h6>, <p>, <div>, <span>

<!-- Links & Images -->
<a href="...">, <img src="..." alt="...">

<!-- Lists -->
<ul>, <ol>, <li>

<!-- Forms -->
<form>, <input>, <textarea>, <button>, <select>

<!-- Semantic -->
<header>, <nav>, <main>, <section>, <footer>
```

### Essential CSS Properties
```css
/* Text */
color, font-size, font-family, font-weight, text-align

/* Box Model */
width, height, padding, margin, border

/* Background */
background-color, background-image

/* Display */
display, position, float

/* Flexbox */
display: flex, justify-content, align-items
```

### Essential Git Commands
```bash
git init          # Initialize repository
git status        # Check status
git add .         # Stage all files
git commit -m ""  # Commit with message
git log           # View history
git push          # Push to remote
git pull          # Pull from remote
```

## Getting Help

### When You're Stuck

1. **Read error messages**: They often tell you what's wrong
2. **Search Google**: "how to [what you want] html/css"
3. **Check MDN**: Most authoritative source
4. **Ask on Stack Overflow**: Format question well
5. **Use DevTools**: Inspect and debug
6. **Ask your trainer**: They're there to help!

### How to Ask Good Questions

1. **Be specific**: "My navigation bar isn't centered" vs "It doesn't work"
2. **Include code**: Use CodePen or paste formatted code
3. **Describe what you tried**: Show your debugging attempts
4. **Provide context**: Browser, OS, relevant details
